Introduction to AI Coding

Welcome to HK's comprehensive guide on leveraging Artificial Intelligence in your development workflow. This documentation is designed to help you navigate the rapidly evolving landscape of AI-assisted coding efficiently.

Artificial Intelligence is transforming how we write, debug, and maintain code. At HK, we believe in empowering our developers with cutting-edge tools. This guide covers everything from setting up your environment to understanding the ethical implications of generated code.

info Up-to-Date Information

This guide is updated weekly to reflect the newest advancements in LLM technology (Large Language Models) and coding assistants tailored for HK's tech stack.

Core Concepts

Before diving into the tools, it is crucial to understand the fundamental concepts that drive AI coding assistants. These systems rely on probabilistic models to predict and generate code snippets.

Key Terminology

  • Large Language Models (LLMs): Deep learning algorithms that can recognize, summarize, translate, predict, and generate text and other content based on knowledge gained from massive datasets.
  • Context Window: The amount of text (code) the AI can consider at one time. A larger context window allows the AI to "understand" more of your project structure.
  • Prompt Engineering: The art of crafting inputs (prompts) to get the best possible output from an AI model.

Setting Up Your Environment

To get started with AI-assisted coding at HK, you will need to configure your development environment. We primarily use VS Code with specific extensions.

Installation Steps

  1. Install Visual Studio Code Ensure you have the latest stable build of VS Code installed on your machine.
  2. Authenticate with HK GitHub Enterprise Log in using your organization credentials to access the licensed AI features.
  3. Install the Copilot Extension Navigate to the Extensions marketplace, search for "GitHub Copilot", and click install.
